兩百字內的個人簡介。
我是張辰宇，是個外表柔和，內在充滿激情的處女座，24歲，畢業於萬能科技大學企管系，前份工作任職於泰豐輪胎儲運課，主要業務為倉儲管理，工作七年八個月，在公司我導入了許多現代化軟體，利用google試算表取代紙本手寫的儲位表，以及利用Plnner 5D重整儲位，直到近期公司被美國課重稅，加班時數減少，收入減少許多，並有了想換工作的念頭，下班回到家，坐在車上的我捫心自問，我待的產業未來會不會被自動化倉儲取代，於是萌生出想轉職的意念。
為了成為軟體工程師，曾做過什麼努力？若有具體專案作品請分享給我們。
今年二月我在Coursera買了Programming for Everybody (Getting Started with Python)這堂課，一開始完全聽不懂，我的英文能力很差，搭配著google翻譯看了很多次才知道什麼是字串，什麼是變數，什麼是資料型態，在當我懷疑是否能轉職勝任成軟體工程師時，YouTube的演算法推薦了彭彭老師的python課程，於是我勤作筆記，將彭彭老師上課說的跟Coursera課堂上的做combine，不但順利讓我拿到證書，還順道拿到了Crash Course on Python這門課的證書。 證書1 證書2 努力勤作筆記1 努力勤作筆記2
如果參與這個計畫，會怎麼安排學習時間？
課程開始前，我第一目標是把CSS Flex box跟Responsive Web Design學好，切版真的是一門學問，怎麼切都切不好，padding、margin都還需要磨練一下。
課程開始後，每個禮拜一早上10點參加線上會議，並遵從老師的指導，並努力提供大於三次以上的每日進度報告，希望利用進度報告來發現我的各項缺失及不足，讓我可以隨時隨地於調整自身學習狀況。
是否有想要加入的軟體公司？為什麼想加入該公司？
我想加入amazing talk show或hahow這一類的教育學習平台，在工作的過程了解學習平台需要如何運作外，下班時間利用公司的教學資源來幫自己充電，讓自己能夠進一步成長，尤其英文是我的軟肋。
請描述一件你曾經碰到最困難的事情，如何克服？
Crash Course on Python這門課最後一個作業是wordcloud，當我把程式碼打完之後，按下輸出，卻什麼東西都沒跑出來，也沒有任何紅字或錯誤碼，我懷疑是不是自己的程式碼打錯，並請求朋友幫忙，朋友po了一段程式碼給我試，wordcloud就是出不來，卡關卡了一個禮拜，google了幾個關鍵字，剛好在stackoverflow看到有人提問，才知道記事本裡的文字只能打英文，並不支持中文，於是修改好後，就順利產出人生第一個wordcloud，因此我也學習到google的重要性。 卡關卡很久的WordCloud
關於這份申請網頁，請分享一個你開發時的技術心得。
這是我第一次自己開發出來的網頁，內心其實蠻激動的，body裡font-family是從google fonts抓的，透過link導入到CSS裡。上面的導航欄則利用老師教的CSS Flex box，logo的部分卡了半天，一直沒辦法變成白色，google搜尋了一下發現可以用CSS Filters其中的brightness(亮度)做調色，我還有做簡單的RWD切版，不然用手機板網頁會跑掉，最後再加上我以前很喜歡無名小站的CSS變更鼠標，換上WeHelp的logo，大致上就是這次作業的技術心得。
其他想要對我們說的事情？
我非常希望能透過這次轉職，未來提供有能力卻沒有錢上課的學員，如果寫程式能改善他們的生活，其實說真的也算是另一種help。非常謝謝彭彭老師，用不到三分之一的價格，幫助我們轉職成軟體工程師，也希望能夠成為您軟體工程師扶持計畫中其中一名助教。
